Cancer correlation & prediction

In this project, I used a dataset about cancer cells. I did a data cleaning, exploration and correlation of the features of the dataset giving some visualization results. Then, using supervised machine learning, I made a linear regression model about the perimeter mean of the cancer cells and a classification model in order to predict if a cancer is benign or malignant. Finally, I made a clustering model using unsupervised machine learning.

This project was made by Python (Jupyter Notebook) and "pandas", "seaborn" and "sklearn" as main libraries.
